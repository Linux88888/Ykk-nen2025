## Prediction Model

The prediction uses a statistical model that considers:

- Actual results from the early matches of the 2025 season
- PK-35's 2-point deduction
- Team strength ratings based on actual squad compositions
- Individual player details from Transfermarkt and scouting reports
- Home advantage factors
- Form/momentum indicators
- Poisson distribution for realistic score modeling
- Community consensus from Futisforum2 for validation

## Data Sources

- Current player performance statistics
